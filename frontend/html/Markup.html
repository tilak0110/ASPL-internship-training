Marking up Computer Code in HTML

Section 34.1: Block with <pre> and <code>

Use this when indentation and formatting matter, like Python code:


x = 42
if x == 42:
    print("x is 42")

Example showing HTML code (special characters must be escaped):

<p>This is a paragraph.</p>

Section 34.2: Inline with <code>

Use the <code> element inline for small snippets like:
<a>, <img>, or console.log()

Example sentence: The a element creates a hyperlink.
